# AWS Status RSS Feeds

Auto-updated list of every AWS Service Health Dashboard RSS feed URL, refreshed daily via GitHub Actions.

Currently tracking **5768 feeds** across **268 services**.

## Files

- `aws_rss_feeds.txt` - flat list, one URL per line
- `aws_rss_feeds.json` - grouped by service name and region

## Source

Feed URLs are derived from the `services.json` endpoint that powers the official [AWS Health Dashboard](https://health.aws.amazon.com/health/status).
